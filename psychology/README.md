# 개요 

이번 월간 데이콘의 심리 성향 예측 경진대회는 마키아밸리즘 심리 테스트를 활용하여 테스트 참가자의 국가 선거 투표 여부를 예측하는 대회입니다.

주어진 데이터는 train 데이터 기준 78개의 칼럼이 있으며 'education' 등 문자열이 포함된 변수도 있습니다.
